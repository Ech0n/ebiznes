## How to run:
1. `docker compose build`
2. `docker compose up` - Serwer uruchamiony jest dopiero jak zobaczymy komunikat Echo o wystartowaniu serwera.
3. Strona jest teraz dostępna pod adresem [http://localhost](http://localhost)
